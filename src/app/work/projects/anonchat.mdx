## Overview

A real-time, privacy-focused anonymous messaging application for secure group chats and discussions. Built using modern web socket protocols to deliver instantaneous messages without retaining private identifiers.

## Key Features

- **No Account Needed**: Start messaging immediately without typing in emails or phone numbers.
- **Real-Time Synchrony**: Instant messaging delivery via secure WebSockets.
- **Privacy First**: Zero message caching on external databases, keeping chat streams temporary.
