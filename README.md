# Devthon Labs — Web & App Development Studio Portfolio

This repository contains the source code for the official portfolio and website of **Devthon Labs**, a web and app development studio specializing in custom full-stack builds, AI integration, and high-performance websites for local businesses.

## Features

- **Project Showcase**: Built-in portfolio sections showcasing work (such as Demo HMS, AnonChat, and StudentCare) with rich MDX support.
- **Blog Engine**: MDX-based blogging platform for sharing developer insights, AI applications, and tech tips.
- **Modern Tech Stack**: Built with [Next.js](https://nextjs.org/) and [Once UI](https://once-ui.com) for clean, fast rendering and consistent design tokens.
- **SEO Ready**: Automatic Open Graph / metadata generation.
- **Fully Responsive**: Optimized layout for all devices and screen sizes.

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run the development server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 3. Build for production
```bash
npm run build
```

## Project Structure

- `src/app/` - Next.js App Router pages (Home, About, Blog, Work).
- `src/components/` - React components for layout, navigation, footer, and features.
- `src/resources/` - Content configuration (`content.tsx`), styles, and design system config.
- `src/types/` - TypeScript interface and type definitions.
- `public/` - Static assets including images, icons, and logos.
